# Node.js Backend Project

A pure Node.js backend project without any external frameworks.

## Getting Started

### Prerequisites
- Node.js (v14 or higher)

### Installation
1. Clone this repository
2. Install dependencies:
```bash
npm install
```

### Running the Server
- Development mode (with auto-reload):
```bash
npm run dev
```

- Production mode:
```bash
npm start
```
